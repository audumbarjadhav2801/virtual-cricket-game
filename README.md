# 🏏 Virtual Cricket Game

A console-based Virtual Cricket Match Simulator developed using **C++** and **Object-Oriented Programming (OOP)** concepts.

## 📌 Overview

This project simulates a cricket match where users create two teams, perform a toss, play innings, and view live score updates. It demonstrates OOP principles and modular programming using multiple source and header files.

## ✨ Features

- Team selection from a pool of players
- Toss simulation
- Batting and bowling simulation
- Live score updates
- Innings management
- Console-based gameplay
- Modular code using classes and header files

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- Header Files
- Modular Programming

## 🚀 How to Run

### Compile

```bash
g++ virtual-cricket-game.cpp game.cpp player.cpp team.cpp -o CricketGame
```

### Run

```bash
./CricketGame
```
